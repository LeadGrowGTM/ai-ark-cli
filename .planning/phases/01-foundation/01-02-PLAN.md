1. **src/client/rate-limiter.ts** — Multi-tier token bucket rate limiter:
   - Export class `RateLimiter`
   - Constructor takes array of `{ maxTokens: number; refillRate: number; refillIntervalMs: number }`
   - Default tiers: `[{ maxTokens: 5, refillRate: 5, refillIntervalMs: 1000 }, { maxTokens: 300, refillRate: 300, refillIntervalMs: 60000 }, { maxTokens: 18000, refillRate: 18000, refillIntervalMs: 3600000 }]`
   - `async acquire()` method: checks all tiers have tokens, if any tier empty waits for the soonest refill, decrements all tiers on success
   - Each tier tracks: `tokens: number`, `lastRefill: number`
   - Refill logic: on each acquire call, calculate elapsed time since lastRefill, add `Math.floor(elapsed / refillIntervalMs) * refillRate` tokens (capped at maxTokens), update lastRefill

2. **src/client/http.ts** — Typed HTTP client:
   - Export class `AiArkClient`
   - Constructor: takes `apiKey: string` (validated non-empty, throws if missing), creates RateLimiter instance
   - Private `baseUrl`: `"https://api.ai-ark.com/api/developer-portal/v1"`
   - Private `headers` getter: returns `{ "Content-Type": "application/json", "X-TOKEN": this.apiKey }`
   - `async get<T>(endpoint: ApiEndpoint): Promise<T>` — awaits rate limiter, sends GET with fetch(), handles errors
   - `async post<T>(endpoint: ApiEndpoint, body: unknown): Promise<T>` — awaits rate limiter, sends POST with JSON body, handles errors
   - Private `handleResponse<T>(response: Response): Promise<T>` method:
     - If response.ok: return response.json() as T
     - If not ok: try to parse body as ApiError, throw an `AiArkApiError` with status, error message, path, and timestamp
     - If body parse fails: throw generic error with status code and statusText
   - Export class `AiArkApiError extends Error`:
     - Properties: `status: number`, `apiError: string`, `path: string`, `timestamp: string`
     - Constructor formats message as: `"AI Ark API Error (${status}): ${apiError}" + (path ? " [${path}]" : "")`
   - Export factory function `createClient(): AiArkClient`:
     - Reads `process.env.AI_ARK_API_KEY`
     - If not set, throws Error with message: `"AI_ARK_API_KEY environment variable is not set. Get your key at https://app.ai-ark.com/settings/api-management/dashboard"`
     - Returns new AiArkClient(apiKey)
